Projet JavaScript : JS-TasksFlow

Application To‑Do riche en fonctionnalités - tri, filtres, export/import.

Retour à l'accueil

Description

JS-TasksFlow est une application légère et intuitive pour gérer vos tâches. Elle permet de définir des dates d'échéance, des priorités et des catégories. De plus, elle offre des fonctionnalités d'export/import au format JSON, tout en intégrant des animations FLIP et des confettis.

Aperçu

Technologies utilisées pour JS-TasksFlow

Frontend

  • HTML5 : structure de base de l'interface utilisateur
  • CSS3 : styles et animations de l'interface
  • JavaScript : logique côté client et interactions

Framework Desktop

  • Electron - Version : ^32.2.0 - permet de transformer l'application web en application desktop et d'intégrer nativement le système d'exploitation Windows

Build & Packaging

  • electron-builder - Version : ^25.1.8 - outil de build pour créer l'exécutable Windows, génère l'installateur NSIS et crée le package distributable

Stockage

  • localStorage : stockage local des données côté client
  • JSON : format de données utilisé pour la persistance et l'export/import

Autres caractéristiques

  • Architecture modulaire
  • Interface native Windows grâce à Electron
  • Gestion des événements système (notifications, raccourcis, etc.)
  • Auto-packaging avec NSIS via electron-builder

Fonctionnement

Ajoutez, éditez, marquez comme terminé ou supprimez des tâches. Filtrez par statut ou catégorie et triez par date ou priorité. Exportez vos tâches au format JSON pour les sauvegarder ou les partager, et importez-les facilement. L'interface utilise des animations FLIP pour des transitions fluides, et des confettis célèbrent chaque tâche terminée.
